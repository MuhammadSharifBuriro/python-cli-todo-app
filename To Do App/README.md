#  CLI ToDo App

A simple Command Line Interface (CLI) based ToDo Application built using Python.

This project helps users manage their daily tasks by adding, viewing, completing, and deleting tasks. Data is stored permanently using JSON file handling.

---

🚀 Features

Add new tasks  
View all tasks  
Mark tasks as completed  
Delete tasks  
Save tasks permanently  
Load tasks automatically when the application starts  
Simple and user-friendly CLI interface

---

##  Technologies Used

- Python 3
- JSON
- File Handling
- Modular Programming
